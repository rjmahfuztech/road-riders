### Features

1. Built user interface using React.js and loaded data from back-end API.
2. Admin and user's dashboard are different.
3. An admin can add a new service, manage the service, and also make a new admin via email.
4. An authenticated user can order service after payment and can see what he ordered.
5. Users can give us a review if he/her satisfied.

### Front-End Technology

- React.js
- React Router
- JavaScript (ES6)
- CSS3
- Bootstrap

### Back-End technology

- Node.js
- Express.js
- MongoDB(database)
- Firebase Authentication
- Heroku
